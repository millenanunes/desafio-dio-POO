#  Bootcamp Java POO

## Sobre o projeto

Projeto desenvolvido durante o desafio de Programação Orientada a Objetos da DIO, com o objetivo de praticar conceitos fundamentais da programação orientada a objetos utilizando Java.

O sistema simula um ambiente de bootcamp, onde desenvolvedores podem se inscrever em conteúdos, progredir nos estudos e acumular experiência (XP).

---

## Conceitos aplicados

- Abstração  
- Encapsulamento  
- Herança  
- Polimorfismo  
- Collections (Set, LinkedHashSet)  
- Stream API  
- Optional
